# 🥇 Gold Layer - Business & Analytics Layer

## 📌 Overview
The Gold Layer contains business-ready data modeled into fact and dimension tables, optimized for reporting and analytics.

---

## 🧱 Data Model

### Fact Tables
- FactListings

### Dimension Tables
- DimHost
- DimLocation
- DimProperty
- DimDay_type

---

## 🔗 Relationships
- FactListings is connected to all dimension tables using foreign keys
- Star Schema is used for optimal performance

---

## 📊 Usage

This layer is directly connected to Power BI to build dashboards and generate insights such as:
- Pricing trends
- Listing distribution
- Superhost analysis
- Location-based insights

---

## 🎯 Purpose

- Enable fast and efficient analytics
- Provide a single source of truth for reporting
- Support business decision-making

---

## 📌 Notes

- Data is fully cleaned and structured
- Optimized using indexing 
- Ready for visualization and BI tools
