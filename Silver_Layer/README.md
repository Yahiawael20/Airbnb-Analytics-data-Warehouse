# 🥈 Silver Layer - Data Cleaning & Transformation

## 📌 Overview
The Silver Layer is responsible for cleaning and transforming raw data from the Bronze layer into a structured and consistent format.

---

## 🔄 Transformations Applied

- Removed null and invalid values
- Standardized column names
- Converted data types (e.g., price, dates)
- Removed duplicates
- Applied basic business rules
- Created derived columns where needed

---

## 📂 Data Flow
Bronze Layer → Silver Layer

---

## 🎯 Purpose

- Improve data quality
- Ensure consistency across datasets
- Prepare data for building analytical models in the Gold layer

---

## 📌 Notes

- No business aggregation is done here
- Data is cleaned but still detailed (not summarized)
- This layer acts as a bridge between raw data and analytics-ready data
